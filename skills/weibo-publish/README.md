# Weibo Publisher Skill

🎯 Automated Weibo publishing with browser automation

## Features
- ✅ Browser automation via CDP
- ✅ Hot search content generation
- ✅ Quality content validation
- ✅ Error handling & recovery
- ✅ Complete documentation

## Quick Start
1. Start VNC: `vncserver :1`
2. Start browser: `DISPLAY=:1 /snap/bin/chromium --remote-debugging-port=18800 https://m.weibo.cn/compose`
3. Test: `python3 scripts/weibo_publish_test.py`
4. Publish: `python3 scripts/weibo_hot_search_publish.py`
