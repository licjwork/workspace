# 微博发布检查清单

## 🟢 发布前检查

### 环境准备
- [ ] VNC服务已启动 (`vncserver :1`)
- [ ] Chromium浏览器已启动
- [ ] 浏览器已登录微博账号
- [ ] CDP调试端口(18800)正常

### 内容准备
- [ ] 内容长度200-300字
- [ ] 包含热门话题
- [ ] 添加相关话题标签
- [ ] 语言生动有趣
- [ ] 检查无敏感内容

## 🟡 发布过程

### 步骤1: 导航到发布页面
- [ ] 确认页面URL: `https://m.weibo.cn/compose`
- [ ] 页面加载完成

### 步骤2: 输入内容
- [ ] 找到文本框（占位符："分享新鲜事…"）
- [ ] 成功输入内容
- [ ] 触发input和change事件

### 步骤3: 发布微博
- [ ] 找到发布按钮（类名：`m-send-btn`）
- [ ] 成功点击发布按钮
- [ ] 等待发布完成

## 🟢 发布后验证

### 页面状态
- [ ] 页面正常跳转
- [ ] 无错误提示
- [ ] 内容已成功发布

### 内容验证
- [ ] 检查个人主页确认发布成功
- [ ] 内容显示正常
- [ ] 话题标签可点击

## 🔧 故障排除

### 常见问题
- **无法输入内容**: 检查文本框是否正确识别
- **发布按钮灰色**: 确认已输入内容
- **跳转到登录页**: 检查登录状态
- **发布失败**: 检查网络连接

### 解决方案
1. 重启浏览器
2. 重新登录微博
3. 检查网络连接
4. 使用测试脚本验证功能
