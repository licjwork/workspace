# 微博发布快速操作指南

## 环境准备

### 1. 启动VNC服务
```bash
vncserver :1 -geometry 1280x800 -depth 24
xhost +
```

### 2. 启动浏览器
```bash
DISPLAY=:1 /snap/bin/chromium \
  --no-sandbox \
  --remote-debugging-port=18800 \
  --remote-allow-origins=* \
  --disable-gpu \
  --no-first-run \
  --disable-default-apps \
  https://m.weibo.cn/compose
```

## 发布流程

### 步骤1: 输入微博内容
**目标元素**: 文本框（占位符："分享新鲜事…"）
**操作**: 输入要发布的微博内容

### 步骤2: 查找发布按钮
**目标元素**: 发布按钮（类名：`m-send-btn`，文本："发送"）
**特征**: `<a class="m-send-btn">发送</a>`

### 步骤3: 点击发布
**操作**: 点击发布按钮完成发布

## 关键元素信息

| 元素类型 | 选择器 | 特征 |
|---------|--------|------|
| 文本框 | `textarea[placeholder*="分享新鲜事"]` | 占位符："分享新鲜事…" |
| 发布按钮 | `a.m-send-btn` | 类名：m-send-btn，文本："发送" |

## 注意事项

1. **登录状态**: 确保已登录微博账号
2. **内容要求**: 必须输入文本内容才能发布
3. **网络连接**: 确保网络连接正常
4. **浏览器状态**: 使用非headless模式的chromium浏览器

## 验证标准

✅ 成功标准：
- 文本框成功输入内容
- 发布按钮成功点击
- 页面跳转或发布成功提示

❌ 失败情况：
- 跳转到登录页面（未登录）
- 按钮点击无响应（内容未输入）
- 网络错误提示
