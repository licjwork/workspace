# 微博发布技能

微博发布技能是一个基于AI生成的高质量微博发布系统，专注于生成真实、有温度的个人感受内容。

## 重大优化

**最新版本特色**：
- 简化逻辑，专注生成真实个人感受
- 第一人称视角，增强代入感和真实性
- 300-500字内容，符合用户阅读习惯
- 包含完整要素：🔥📊💡⚠️❤️#️⃣

## 使用方法

使用命令: python3 skills/weibo-publish/scripts/smart_weibo_publisher.py --topic "话题关键词"

**内容特点**：
- 真实个人感受，非模板化内容
- 第一人称"我"的视角讲述
- 情感丰富，有温度有共鸣
- 结构完整，包含所有必要元素
- 字数适中，300-500字
