# 微博发布技能

微博发布技能是一个基于AI生成的高质量微博发布系统，专注于生成真实、有温度的个人感受内容。

## 重大优化

**最新版本特色**：
- 简化逻辑，专注生成真实个人感受
- 第一人称视角，增强代入感和真实性
- 300-500字内容，符合用户阅读习惯
- 包含完整要素：🔥📊💡⚠️❤️#️⃣

## 使用方法

使用命令: python3 skills/weibo-publish/scripts/smart_weibo_publisher.py --topic "话题关键词"

**内容特点**：
- 真实个人感受，非模板化内容
- 第一人称"我"的视角讲述
- 情感丰富，有温度有共鸣
- 结构完整，包含所有必要元素
- 字数适中，300-500字

## 技术实现

### 前置依赖
微博发布技能依赖以下服务正常运行：
1. **VNC服务器** - 提供浏览器显示环境
2. **Chromium浏览器** - 运行微博发布页面
3. **CDP接口** - 浏览器调试协议，用于自动化操作

### 有效发布步骤

1. **服务检查**
   - 检查VNC和浏览器进程是否运行
   - 验证CDP接口是否可连接
   - 确保微博发布页面已加载

2. **内容生成**
   - 调用AI生成系统创建高质量内容
   - 内容包含：标题、数据观察、要点分析、实用建议、情感共鸣、话题标签
   - 自动适配不同话题类型（科技、安全、经济、教育、旅游等）

3. **内容输入**
   - 使用CDP协议连接浏览器
   - 定位微博输入框：`textarea[placeholder*="分享新鲜事"]`
   - 设置输入框内容并触发input事件
   - 验证内容是否成功设置

4. **发布执行**
   - 等待2秒确保内容稳定
   - 定位发布按钮：`a.m-send-btn`
   - 检查按钮是否可用（无disabled类）
   - 点击发布按钮
   - 验证发布是否成功

### 故障排除

**常见问题**：
- **输入框定位失败** - 检查微博页面是否正确加载，placeholder属性是否匹配
- **发布按钮点击失败** - 确保内容已正确输入，按钮未被禁用
- **CDP连接失败** - 检查浏览器调试端口(18800)是否开放

**解决方案**：
- 重启VNC和浏览器服务
- 检查浏览器启动参数是否正确
- 验证微博账号登录状态
