# OpenClaw定时任务管理

## 智能新闻推送
- **任务**: 白天每2小时执行完整的智能新闻推送流程
- **触发条件**: 检测到当前时间在7:00-23:00之间且小时数为偶数（8点、10点、12点、14点、16点、18点、20点、22点）
- **执行流程**: 
  1. 搜索英文新闻 → 2. DeepSeek高质量翻译 → 3. 编辑news.json文件 → 4. 飞书推送
- **新闻数量**: 10条
- **内容长度**: 每条摘要约100字
- **命令**: `python3 scripts/intelligent_news_simple.py`
- **输出文件**: `/home/ubuntu/.openclaw/workspace/web-navigation-system/database/news.json`
- **注意**: 避开睡眠时间23:00-7:00，确保推送不打扰休息

## 服务状态监控
- **任务**: 每小时检查导航系统服务状态
- **触发条件**: 每整点执行一次
- **执行**: 检查前后端服务运行状态，异常时自动重启

## 数据库备份
- **任务**: 每天23:00备份数据库
- **触发条件**: 23:00-23:30之间
- **执行**: 备份SQLite数据库到指定目录
