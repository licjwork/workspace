## 重要规则 - 文件读取规范

### 正确做法 ✅
- 使用 `exec` 工具配合 Linux 命令读取本地文件
- 常用命令：`cat`、`head`、`tail`、`ls` 等
- 示例：`exec(command="cat filename.txt", workdir="path/to/directory")`

### 禁止做法 ❌
- 禁止使用飞书 read 工具读取本地文件
- 这会导致报错和错误操作

### 适用范围 ⚠️
- 此规则适用于所有本地文件系统操作
- 包括工作目录中的所有文档和文件

### 例外情况 ✅
- 只有当读取飞书云文档时，才可以使用飞书 read 工具
- 本地文件一律使用 exec + Linux 命令方式
